# Sanity Plugins Consolidated

This file consolidates content from the Sanity.io Plugins directory (https://www.sanity.io/plugins) as of December 26, 2025. The plugins page serves as a community-driven directory for extending Sanity Studio with custom inputs, tools, asset sources, dashboards, integrations, and more. Plugins are contributed by the official Sanity team and the community. The page features sections like "Must-haves", "Clients & SDKs", "Portable Text", "Custom inputs", "Latest tools", and others. Many plugins have individual detail pages at https://www.sanity.io/plugins/[slug].

The directory includes hundreds of plugins (exact total not specified, but searchable and paginated). Below is a structured list of prominent and featured plugins extracted from the main page and related searches, including names, short descriptions, authors (where available), categories, and links.

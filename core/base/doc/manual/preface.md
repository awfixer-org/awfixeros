# Preface {#preface}

This manual describes how to install, use and extend awos, a Linux distribution based on the purely functional package management system [Nix](https://awos.org/nix), that is composed using modules and packages defined in the [Nixpkgs](https://awos.org/nixpkgs) project.

Additional information regarding the Nix package manager and the Nixpkgs project can be found in respectively the [Nix manual](https://awos.org/nix/manual) and the [Nixpkgs manual](https://awos.org/nixpkgs/manual).

If you encounter problems, please report them on the [`Discourse`](https://discourse.awos.org), the [Matrix room](https://matrix.to/#/%23nix:awos.org), or on the [`#awos` channel on Libera.Chat](irc://irc.libera.chat/#awos). Alternatively, consider [contributing to this manual](#chap-contributing). Bugs should be reported in [awosâ€™ GitHub issue tracker](https://github.com/awos/nixpkgs/issues).
